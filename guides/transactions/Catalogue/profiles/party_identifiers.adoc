= Party Identifiers

The “schemeID” attribute shall be populated in all instances of the “ID” element when used within a
“PartyIdentification”-container and in all instances of the “EndpointID” element when used within a “Party”-
container. This value will be evaluated against codelists, and only if using values that are in the codelist they will be accepted as valid. 
The codelist used for “EndpointID” is “EAS” (Electronic Address Scheme) and för “ID” in the “PartyIdentification”-container the codelist is “ICD” 
(ISO 6523).
