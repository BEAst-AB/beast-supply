[[parties]]
= Parties

In the pre-award catalogue request you can provide information on the following parties:

Catalogue request provider:: The party that sends the catalogue request. Information on catalogue request provider is mandatory
Catalogue request receiver:: The party that receives the catalogue request. Information on catalogue request receiver is mandatory
